# Javascript : 

Javascript               vs               Java
------------                              ------
Scripting language                        Programming language
Interpreted                               Compiled & Interpreted
Dynamically/loosely typed                 Statically/strictly typed
Object based                              Object Oriented
; is optional                             ; is mandatory
String can be ''/""                       Only in ""

# Javascript Types : 
--------------------
1) Core Javascript --> Can work on both the side, variables, constants, datatypes, operators, operands, control statements, Inbuilt functions, custom functions, inbuilt objects, custom objects, Timer apis, rest and spread, etc.
2) Client side Javascript --> BOM, DOM, Event Handling, etc. (Angular, React, Vue JS)
3) Server side Javascript --> File Handling, DB ops, Middleware etc. (Node, express JS)

# datatypes in js :

3) string

5) object
6) function
